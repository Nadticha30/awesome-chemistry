# เคมีเจ๋ง (Awesome Chemistry) 🧪

**แอปพลิเคชันสำหรับการเรียนรู้ศาสตร์เคมีและการคำนวณทางเคมี**  
โมบายแอปพลิเคชันบนระบบปฏิบัติการ Android (พัฒนาด้วย Android Studio) ที่ออกแบบมาเพื่อเป็นสื่อการเรียนรู้ทางด้านวิชาเคมีสำหรับนักเรียนและผู้สนใจทั่วไป ช่วยให้การเรียนตารางธาตุและการคำนวณสัญลักษณ์นิวเคลียร์กลายเป็นเรื่องง่าย สะดวกรวดเร็ว และสนุกสนานผ่านระบบเกมตอบคำถามอัจฉริยะ

## 🛠️ ส่วนงานที่รับผิดชอบหลัก (Owner Responsibilities)
1.  **หน้าคำนวณสัญลักษณ์นิวเคลียร์ (Nuclear Calculation Screen):** ออกแบบสถาปัตยกรรม UI และเขียนคำสั่งประมวลผลทางคณิตศาสตร์เพื่อหาจำนวนอนุภาคมูลฐานของอะตอม
2.  **หน้าหลักการคิดคำนวณ (Calculation Logic Screen):** พัฒนาหน้าจอสรุปสูตรฟิสิกส์เคมี เพื่อเป็นคู่มือและแหล่งอ้างอิงให้ผู้ใช้งานเข้าใจที่มาของคำตอบ
3.  **เกมจับคู่สัญลักษณ์และชื่อธาตุ (Element Matching Game):** ออกแบบและพัฒนา logic เกมจับคู่การจำสัญลักษณ์ธาตุในตารางธาตุเพื่อเพิ่มความสนุกในการจดจำ
4.  **หน้าจบเกมและการส่งต่อระบบ (Victory/Game Over Screen):** สร้างจุดสิ้นสุดของเกมพร้อมแสดงผลคะแนนและประสิทธิภาพของผู้เล่น ก่อนส่งต่อข้อมูลระบบให้กับเพื่อนในทีมเพื่อบูรณาการเข้ากับส่วนอื่น

## 💻 Tech Stack
*   **IDE / Platform:** Android Studio (ระบบปฏิบัติการ Android)
*   **Programming Language:** Java / Kotlin
*   **UI/UX Design Tools:** Canva (สำหรับออกแบบทรัพยากรภาพ ปุ่ม หน้าจอ และพื้นหลังแอปพลิเคชัน)
*   **Application Framework:** Android Edge-to-Edge & AppCompat
*   **Landing Page Tech:** HTML5, CSS3 (Vanilla CSS with Glassmorphism & Cyber Theme), JavaScript (ES6)

## 📸 Project Workflow
*   **Phase 1: Nuclear Calculation UI & Core Function** (ระบบคำนวณและหน้าคำนวณ)
*   **Phase 2: Implementation of Calculation Formulas** (หน้าแสดงหลักการคิดคำนวณ)
*   **Phase 3: Interactive Element Matching Game** (ระบบเกมจับคู่สัญลักษณ์และชื่อธาตุ)
*   **Phase 4: Game Over & Handover Process** (การพัฒนาหน้าจบเกมและส่งต่อโมดูล)
