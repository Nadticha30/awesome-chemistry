My Responsibilities

งานที่ฉันรับผิดชอบหลัก

ในโปรเจกต์นี้ ฉันรับผิดชอบในส่วนการพัฒนาอินเตอร์เฟส ฟังก์ชันการคำนวณทางวิทยาศาสตร์ และระบบมินิเกมการเรียนรู้

1. คำนวณสัญลักษณ์นิวเคลียร์

ออกแบบสถาปัตยกรรม UI และเขียนคำสั่งประมวลผลทางคณิตศาสตร์เพื่อหาจำนวนอนุภาคมูลฐาน (โปรตอน, นิวตรอน, อิเล็กตรอน) ของอะตอมจากเลขมวล เลขอะตอม และประจุ

2. แสดงหลักการคิดคำนวณ

พัฒนาหน้าจอสรุปสูตรและหลักเกณฑ์ทางฟิสิกส์เคมี เพื่อแสดงที่มาของตรรกะเบื้องหลังการทำงานและให้ผู้ใช้งานเข้าศึกษาโครงสร้างสูตรการคำนวณย้อนหลังได้

3. เกมจับคู่สัญลักษณ์และชื่อธาตุ

ออกแบบและเขียนระบบ Matching Game สุ่มจับคู่สัญลักษณ์ธาตุและชื่อธาตุ พร้อมระบบคอยดึงฐานข้อมูลมาแสดงรายละเอียด ข้อมูลเฉพาะ คาบ หมู่ และน้ำหนักอะตอมเมื่อจับคู่ถูกต้อง

4. หน้าสรุปผลและส่งต่อระบบ

สร้างจุดสิ้นสุดของเกมพร้อมแสดงผลคะแนนและประสิทธิภาพของผู้เล่น (Victory / Game Over Screens) พร้อมทำระบบส่งต่อตัวแปรผลลัพธ์เพื่อทำงานร่วมกับฐานข้อมูลหลักของทีม

Project Workflow

ขั้นตอนการพัฒนาและระบบงาน

เจาะลึกขั้นตอนการออกแบบ UI และพัฒนาซอร์สโค้ดในส่วนต่างๆ ที่ได้รับมอบหมายใน 4 ระยะหลัก

01
Phase 1 ระบบคำนวณนิวเคลียร์
02
Phase 2 หน้าแสดงหลักการคิดสูตร
03
Phase 3 เกมจับคู่สัญลักษณ์ธาตุ
04
Phase 4 หน้าจบเกมและส่งต่อข้อมูล
Phase 1: Nuclear Calculation UI & Core Function

ระบบคำนวณและหน้าคำนวณสัญลักษณ์นิวเคลียร์

ออกแบบหน้าจอคำนวณอนุภาคมูลฐานของสัญลักษณ์นิวเคลียร์ โดยรับข้อมูลนำเข้า (Input) ได้แก่ เลขมวล, ประจุ และเลขอะตอม จากนั้นพัฒนาส่วนเขียนโปรแกรม (Source Code) เพื่อประมวลผลหาจำนวนของ โปรตอน นิวตรอน และอิเล็กตรอน พร้อมแสดงผลลัพธ์คำตอบทางหน้าจอแบบอัตโนมัติ

Phase 2: Implementation of Calculation Formulas

หน้าแสดงหลักการคิดคำนวณและสูตรเคมี

การพัฒนาหน้าจอแสดงสัจพจน์และสูตรที่ใช้คำนวณทางเคมี เพื่อแสดงความโปร่งใสของ Logic การประมวลผลหลังบ้าน โดยเขียนโค้ดและพัฒนา UI เพื่อแสดงหลักการคำนวณอนุภาคต่างๆ ดังนี้

จำนวนโปรตอน
p = Z (เลขอะตอม)
จำนวนนิวตรอน
n = A − Z (เลขมวล − เลขอะตอม)
จำนวนอิเล็กตรอน
e = p ± ประจุ (โปรตอน ± ประจุสุทธิ)
Phase 3: Interactive Element Matching Game

ระบบเกมจับคู่สัญลักษณ์และชื่อธาตุ

ออกแบบและเขียนฟังก์ชันการควบคุมกิจกรรมเกมจับคู่ (Matching Game) โดยสร้างระบบเลือกการจับคู่สัญลักษณ์ธาตุให้ตรงกับชื่อธาตุ พร้อมดึงฐานข้อมูลมาแสดงรายละเอียด ข้อมูลจำเพาะ คาบ หมู่ และน้ำหนักอะตอมของธาตุนั้น ๆ เมื่อผู้เล่นจับคู่ได้ถูกต้อง เพื่อช่วยเพิ่มทักษะความจำและการเรียนรู้ตารางธาตุ

Phase 4: Game Over & Handover Process

การพัฒนาหน้าจบเกมและระบบส่งต่อตัวแปร

พัฒนาระบบสรุปผลลัพธ์การเล่นเกมและแสดงหน้าจอชัยชนะ/จบเกม (Victory Screen) เพื่อแสดงคะแนนความสำเร็จของผู้เล่น จำนวนข้อที่ถูกต้องและเวลาที่ใช้ พร้อมทำระบบส่งต่อตัวแปรผลลัพธ์ (Data Handover) และฟังก์ชันเพื่อประสานร่วมกับระบบฐานข้อมูลหลักและเมนูหลักของเพื่อนทีมวิศวกรรมซอฟต์แวร์ได้อย่างสมบูรณ์
