เคมีเจ๋ง (Awesome Chemistry) 
Application ส ำหรับกำรเรียนรู้ศำสตร์เคมีและกำรค ำนวณทำงเคมี 
"เคมีเจ๋ง" คือโมบายแอปพลิเคชันบนระบบปฏิบัติการ Android (พัฒนาด้วย Android Studio) 
ที่ออกแบบมาเพื่อเป็นสื่อการเรียนรู้ทางด้านวิชาเคมีส าหรับนักเรียนและผู้สนใจทั่วไป 
ช่วยให้การเรียนตารางธาตุและการค านวณสัญลักษณ์นิวเคลียร์กลายเป็นเรื่องง่าย สะดวกรวดเร็ว 
และสนุกสนานผ่านระบบเกมตอบค าถามอัจฉริยะ

🛠️ My Responsibilities (ส่วนงำนที่ฉันรับผิดชอบหลัก) 
ในโปรเจกต์นี้ ฉันท าหน้าที่รับผิดชอบในส่วนการพัฒนาอินเตอร์เฟสและฟังก์ชันการค านวณทางวิทยาศาสตร์ 
รวมถึงระบบเกมเพื่อการศึกษาภายในแอปพลิเคชัน ดังนี้: 
1. หน้ำค ำนวณสัญลักษณ์นิวเคลียร์ (Nuclear Calculation Screen): ออกแบบสถาปัตยกรรม UI 
และเขียนค าสั่งประมวลผลทางคณิตศาสตร์เพื่อหาจ านวนอนุภาคมูลฐานของอะตอม 
2. หน้ำหลักกำรคิดค ำนวณ (Calculation Logic Screen): พัฒนาหน้าจอสรุปสูตรฟิสิกส์เคมี 
เพื่อเป็นคู่มือและแหล่งอ้างอิงให้ผู้ใช้งานเข้าใจที่มาของค าตอบ 
3. เกมจับคู่สัญลักษณ์และชื่อธำตุ (Element Matching Game): ออกแบบและพัฒนา logic 
เกมจับคู่การจ าสัญลักษณ์ธาตุในตารางธาตุเพื่อเพิ่มความสนุกในการจดจ า 
4. หน้ำจบเกมและกำรส่งต่อระบบ (Victory/Game Over Screen): 
สร้างจุดสิ้นสุดของเกมพร้อมแสดงผลคะแนนและประสิทธิภาพของผู้เล่น 
ก่อนส่งต่อข้อมูลระบบให้กับเพื่อนในทีมเพื่อบูรณาการเข้ากับส่วนอื่น 
📸 Project Workflow (ส่วนงำนที่ฉันรับผิดชอบหลัก) 
ขั้นตอนการพัฒนาหน้าจอและระบบการค านวณทั้งหมดที่ฉันได้รับมอบหมาย: 
Phase 1: Nuclear Calculation UI & Core Function 
(ระบบค ำนวณและหน้ำค ำนวณ) 
[ แปะรูปภำพที่ 1, รูปภำพที่ 2 และ รูปภำพที่ 10 ตรงนี้ ] 
● ค ำอธิบำยภำพ: ออกแบบหน้าจอค านวณอนุภาคมูลฐานของสัญลักษณ์นิวเคลียร์ โดยรับข้อมูลน าเข้า

Phase 3: Interactive Element Matching Game 
(ระบบเกมจับคู่สัญลักษณ์และชื่อธำตุ) 
[ แปะรูปภำพที่ 5, รูปภำพที่ 6, รูปภำพที่ 9 และ รูปภำพที่ 11 ตรงนี้ ] 
● ค ำอธิบำยภำพ: ออกแบบและเขียนฟังก์ชันการควบคุมกิจกรรมเกมจับคู่ (Matching Game) 
โดยสร้างระบบเลือกการจับคู่สัญลักษณ์ธาตุให้ตรงกับชื่อธาตุ พร้อมดึงฐานข้อมูลมาแสดงรายละเอียด 
ข้อมูลจ าเพาะ คาบ หมู่ และน้าหนักอะตอมของธาตุนั้น ๆ เมื่อผู้เล่นจับคู่ได้ถูกต้อง เพื่อเสริมทักษะความจ า 
Phase 4: Game Over & Handover Process 
(กำรพัฒนำหน้ำจบเกมและส่งต่อโมดูล) 
[ แปะรูปภำพที่ 7 และ รูปภำพที่ 8 ตรงนี้ ] 
● ค ำอธิบำยภำพ: พัฒนาระบบสรุปผลลัพธ์การเล่นเกมและแสดงหน้าจอชัยชนะ/จบเกม (Victory Screen) 
เพื่อแสดงคะแนนความส าเร็จของผู้เล่น พร้อมท าระบบส่งต่อตัวแปรผลลัพธ์ (Data Handover) 
และฟังก์ชันเพื่อประสานร่วมกับระบบฐานข้อมูลหลักและเมนูหลักของเพื่อนทีมวิศวกรรมซอฟต์แวร์ได้อย่าง
